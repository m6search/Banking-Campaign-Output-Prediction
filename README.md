# Banking-Campaign-Output-Prediction
```
The Description can be found through the following links:
1. https://archive.ics.uci.edu/ml/datasets/bank+marketing
2. https://docs.1010data.com/Tutorials/MachineLearningExamples/BankMarketingDataSet.html

Данные связаны с проведением прямых маркетинговых исследований банковского учреждения. 
Данные маркетинговых акции были основаны на телефонных звонках. 
Во многих случаях требовалось более одного контакта с одним и тем же клиентом чтобы выяснить, 
будет ли продукт принят или нет.
```
